This project (Form 26AS TDS Reconciliation System) consists of an end-to-end web application comprising of a frontend application that uses HTML, CSS and vanilla
JavaScript and backend application that uses Python (Flask) and the database used is MySQL which stores all the records of the tables created.

The web application runs on the web and displays a dashboard consisting of 7 screens along with a sidebar for navigation.

To run the application:
Make sure you are connnected to MySQL database with your correct credentials.
Run:
app.py [in the CLI (terminal)].
